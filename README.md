# BeaverHome

## Stack

### Frontend
- React 18
- Vite 5
- TypeScript

### Backend
- Express (TypeScript)
- SQLite

## Setup (npm)

```bash
npm install
```

### Run the frontend (dev)

```bash
npm run dev
```

This starts both the Vite frontend on http://127.0.0.1:5000 and the API server on
http://127.0.0.1:5001. If you only want the frontend, use:

```bash
npm run dev:ui
```

### Build for production

```bash
npm run build
```

### Preview the production build

```bash
npm run preview
```
