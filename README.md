# BeaverHome

## Stack

### Frontend
- React 18
- Vite 5
- TypeScript

### Backend
- Express (TypeScript)
- SQLite

## Setup (npm)

```bash
npm install
```

### Run the frontend (dev)

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview the production build

```bash
npm run preview
```
